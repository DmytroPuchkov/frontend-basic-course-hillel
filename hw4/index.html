Справка о селекторах

Типы селекторов

По имени тега: h1 {}, div {}, h1 {}, p {}, span {}, a {} и тд. В этом примере применен селектор для тега - a (фиолетовый текст в верхнем регистре и увеличили начертание)
По ID: #unique {} Пример оформления с помощью селектора ID - #example-selector-id.
По классу: .classname {} К этому тексту применен селектор с названием класса example-selector-classname (сделали цвет текста голбым и сделали желтую тень)
По нескольким классам: .classname1 {} .classname2 {} .classname3 {} Применение к тексту трех классов: highlight (черны фон), upper (верхний регистр), active (красный цвет)
По тегу и классу: span.classname {} Оформили текст в желтый в теге span, в котором есть класс not-active среди двух других классов
По тегу, классу и ID span.classname{}#id Через span .classname #id поменяли текст на перечеркнутый и серый
